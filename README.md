# Attendance Application

A simple and functional Python attendance tracking application using RFID scanner input.

## Features
- Create new attendance sheets with event details
- Load existing attendance records
- Support RFID scanner and manual student ID input
- Export attendance data to Excel files
- Fast performance for high-volume scanning

## Setup
1. Ensure you have Python 3.7+
2. Install dependencies: `pip install openpyxl`
3. Place `studentinfo.xlsx` in the application directory with:
   - Column A: Student ID
   - Column K: RFID

## Usage
Run `python attendance.py` to start the application.
